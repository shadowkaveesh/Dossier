Spec 01 / Pipeline

From Brief To Useful Output

Every run starts with the inputs that matter, gives the model a clear job, and retains an explicit quality check before handoff.

01 / CONTEXT

Load the signal

Audience, brand rules, reference material, commercial objective, and output format are gathered before the prompt starts doing work.

02 / ITERATE

Draft with checks

Research, draft, critique, and polish are treated as distinct stages so assumptions can be reviewed instead of buried in a one-shot answer.

03 / HANDOFF

Ship with intent

The final response is formatted for its actual destination: campaign, product page, content plan, research brief, or operational document.
